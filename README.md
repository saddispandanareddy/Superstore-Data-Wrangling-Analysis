# 🧹 Task 1: Data Immersion & Wrangling - Superstore Dataset

## 📌 Objective
The objective of this project is to understand, clean, and prepare a real-world dataset for analysis. Data wrangling is an essential first step in any data analytics workflow.

---

## 📊 Dataset
This project uses the **Superstore Dataset**, which contains business-related data such as:

- Customer information
- Product categories
- Sales and profit details
- Geographic data (region, state, city)

---

## 🔍 Steps Performed

### 1. Data Understanding
- Loaded dataset using Python
- Explored structure and column details
- Analyzed dataset using `.head()`, `.info()`, `.describe()`

### 2. Data Quality Assessment
- Checked for missing values
- Identified duplicate records
- Reviewed dataset consistency

### 3. Data Cleaning
- Removed duplicate rows
- Ensured data consistency

### 4. Feature Engineering
- Created a new column:
  - **Profit_Ratio = Profit / Sales**

---

- Pandas
